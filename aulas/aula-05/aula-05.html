Aula 5: Desbravando Objetos e Construindo uma Tabela de Classificação

Apresentação

Nesta aula da Imersão Dev, aprenderemos a criar uma tabela de classificação.

Principais Tópicos

  • Remover o código estático do HTML;
  • Criar um objeto no JavaScript;
  • Exibir o objeto na página HTML;
  • Criar funções e condicionais para preencher a tabela com os dados necessários.
